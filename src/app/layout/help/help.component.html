Frequently Asked Questions

How do I upload an image?

Navigate to the Admin Panel, and you will find a drag-and-drop area or a button to select images for upload.

Can I convert images to different formats?

Yes, the application supports converting images. Look for the 'Convert' option when viewing an uploaded image.

Is my data secure?

We use industry-standard security measures to protect your data, including secure storage and access control policies.
